# 🤖 AI Business Intelligence Agent

An AI-powered data analytics web app built with Python and Streamlit.

## 🚀 Features
- 📁 CSV File Upload
- 🔍 Automatic Data Profiling
- 📈 KPI Dashboard
- 📊 Auto-Generated Charts (6 types)
- 🧹 AI Data Cleaning + Download
- 🤖 Natural Language Q&A
- 📝 AI Business Report + Download

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Groq API (LLaMA 3.3)
